# excel拆分规则
* 固定长度拆分
* 指定符号拆分
* 忽略重复拆分符号
* 带文本识别符号的拆分（比如`"1,2,3,4,5"`现在按照逗号分隔，但是又设置文本识别符号为`"`，那么`"1,2,3,4,5"`会被当做一个整体，不会进一步参与拆分）

# 正则表达式
固定长度直接substr即可，这里不讨论。下面的正则表达式中`{split}`代表拆分符号，`{text}`代表文本识别符号。  

## 指定符号拆分正则：
```regexp
[{split}]
```

## 忽略重复拆分符号
```regexp
[{split}]+
```

## 带文本识别符号的拆分
```regexp
(?![^{text}]+{text}{split}|[^{text}]+{text}$)
```
